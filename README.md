# Drezyna

Drezyna is a simple Rails application template used at Nukomeet.

![Drezyna](http://nukomeet.s3.amazonaws.com/logo.svg)

## Installation

Install it as:

    $ gem install drezyna

## Usage

    drezyna <your name>
